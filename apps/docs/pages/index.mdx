# React Useful Hooks

A collection of useful React hooks for your projects.

## Getting Started

Install the package:

```bash
pnpm add @react-useful-hooks/react-hooks
```

## Available Hooks

- [useToggle](/hooks/use-toggle) - A hook for managing boolean state with toggle functionality
- [useCounter](/hooks/use-counter) - A hook for managing numeric counter state with increment, decrement, and reset functionality
- [unrefElement](/hooks/unref-element) - A utility for safely unwrapping React refs to get DOM elements
